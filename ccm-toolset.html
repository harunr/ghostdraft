CCM Suite Designed for Business and IT users

The GhostDraft CCM Suite is powered by a series of independent software components, delivering advanced functionality and intuitive customer communications management capabilities for both business and IT users.

Get Started

Document Composition Engine

Stable and Flexible Document Automation

GhostDraft’s Document Composition Engine connects content authors to multiple core systems through a user-friendly, logic-based library known as the Document Domain Model, making it easy for business users to create and test templates in real-time.

  • Underpins the whole toolset
  • Document automation
  • Tens of thousands of users
  • Document Domain Model
  • Thousands of customers

Studio

Template Authoring and Data-Capture Design

Developed with the business user in mind, Studio provides an easy-to-use template builder, empowering content authors to generate and test batch, on-demand, and interactive documents without IT.

  • Point & click mark-up
  • Unique Data Domain Model
  • Auto-created data scenarios
  • Natural-language logic
  • Rich data-capture form design
  • Real-time testing

Configuration Manager

Production Server Management

GhostDraft Configuration Manager streamlines communications management, providing an approval workflow that ensures consistency and compliance for customer-facing communications.

  • Publishing or retiring
  • Data Mapping Transforms
  • Controlling user access
  • Template Packages
  • Workflow Plans
  • Producing usage reports

Server & Services

Access to the GhostDraft Production Services

Web-based suite provides access to all production services, delivering the ability to manage customer communications anywhere at any time. Leverage the Server and Services recognize significant productivity improvements through improved communications and coordination.

  • Document Composition
  • Data Capture
  • Rich set of API’s
  • Data Mapping
  • Policy Assembly
  • Flexible and customizable

Prologue

Document Production and Distribution

Power non-technical users with approved templates in a web-based solution with Prologue. Prologue provides business users ability to customize compliant templates and quickly deliver omnichannel communications, creating a better customer experience.

  • Browser-based application
  • Data-capture forms
  • Interactive documents
  • Template libraries
  • External data sources
  • Production and distribution

Data Transform Builder

Through one-time data mapping, IT users can connect data from multiple core systems to GhostDraft’s Document Domain Model designed to provide intuitive access for business users, eliminating IT resources from the template generation process.
